User Manual for Associates

Add New User:

 

 

1.       Click on Add new user button.

2.       Select the Employee type – Analyst , Economist, Sales, Corp Access , Strategist .

3.       If Employee Type is – Analyst , then selecting a Sector is mandatory.

4.       List of sectors will be displayed in the dropdown, and user can select any one sectors from the list.

5.       If same analyst  is assigned to multiple sectors, then user will have to add that user multiple times with different sectors.

For Example :  Analyst A is assigned to sector Auto, Banks and AGRO

Then user will have to add Analyst A separately for each sector. 3 Records will be displayed in the screen for Analyst A .

 

 

 

Edit existing User:

1.       Click on the Edit button given on the right side of the table.

2.       The selected record will be displayed in Editable mode .

3.       Edit the required data as shown in picture below, and click on Save button.

4.       The Record will be Edited and Saved successfully.

 

 

 

Delete Existing User :

 

1.Click on Delete button , given on right side of the table.

2.An Alert message will be displayed , for confirming the Delete.

3.If user selects Yes, then the select record will be deleted from the Database , and Voter will not be able to see that record for Voting.

 

Uploading Excel sheet:

 

1.       If user wants to upload multiple records in a excel , then user can download the Excel Template by clicking on Download Template button.

2.       An excel file named BrokerUploadFormat.xls  will be downloaded in the system,it should get downloaded in your default Downloads folder.

3.       User will open the file, select the Sector from dropdown  box  given in excel, select the Employee type from dropdown , enter other details of employee.

4.       Save the excel sheet in same format and same name ie  BrokerUploadFormat.xls

5.       Make sure that all the entries done in excel are correct.

6.       If any  Analyst is added then Sector should also be selected.

7.       User can not enter any Sector other than specified in the dropdown list.

8.       If 1 analyst is handling multiple sectors , then multiple rows need to be added for that analyst.

9.       Only one Analyst per row will be added.

10.   Only 1 Sector per row will be added.

13 Click on Upload File button.
